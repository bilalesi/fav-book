# Extension Icons

This directory contains the extension icons in various sizes.

Required sizes:

- icon-16.png (16x16) - Toolbar icon
- icon-32.png (32x32) - Toolbar icon @2x
- icon-48.png (48x48) - Extension management page
- icon-128.png (128x128) - Chrome Web Store

## Placeholder Icons

For development, you can create simple placeholder icons or use the following approach:

1. Create a simple SVG icon
2. Export it to PNG at the required sizes
3. Place the files in this directory

The icons should represent a bookmark or collection concept.
