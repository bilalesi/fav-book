# Extension Screenshots Guide

This document provides guidance for creating professional screenshots for the Chrome Web Store and Firefox Add-ons listings.

## Screenshot Requirements

### Chrome Web Store

- **Format:** PNG or JPEG
- **Size:** 1280x800 or 640x400 pixels
- **Count:** Minimum 1, maximum 5
- **File size:** Maximum 5MB each

### Firefox Add-ons

- **Format:** PNG or JPEG
- **Size:** 1280x800 or 1920x1080 pixels (recommended)
- **Count:** 3-5 recommended
- **File size:** Maximum 10MB each

## Screenshot Creation Tips

### General Guidelines

1. **Use Real Content:**

   - Create realistic sample bookmarks
   - Use actual tweets and LinkedIn posts (with permission or mock data)
   - Show varied content types (text, images, videos)

2. **Clean UI:**

   - Close unnecessary browser tabs
   - Hide bookmarks bar
   - Use a clean browser profile
   - Ensure no personal information is visible

3. **Consistent Branding:**

   - Use the same browser for all screenshots
   - Maintain consistent zoom level
   - Use the same color scheme
   - Keep styling consistent

4. **Professional Quality:**

   - Use high resolution (1280x800 minimum)
   - Ensure text is readable
   - No blurry or pixelated images
   - Good contrast and lighting

5. **Highlight Key Features:**
   - Use subtle arrows or highlights
   - Add annotations if helpful
   - Don't overdo visual indicators
   - Keep it clean and professional

### Tools for Creating Screenshots

**Screenshot Capture:**

- macOS: Cmd+Shift+4 (select area) or Cmd+Shift+5 (screenshot tool)
- Windows: Windows+Shift+S (Snipping Tool)
- Linux: Gnome Screenshot or Spectacle
- Browser extensions: Awesome Screenshot, Nimbus Screenshot

**Image Editing:**

- Figma (free, web-based)
- Canva (free tier available)
- GIMP (free, open-source)
- Photoshop (paid)
- Sketch (macOS, paid)

**Adding Annotations:**

- Figma: Add shapes, arrows, text
- Canva: Use elements and text tools
- Skitch: Simple annotation tool
- Annotate: Browser-based tool

### Screenshot Template

Create a template in Figma or Photoshop:

```
Canvas: 1280x800px
Background: #F5F5F5 or white
Border: Optional 1px subtle shadow
Padding: 40px from edges
Title area: Optional 60px at top for caption
```

### Example Workflow

1. **Prepare the environment:**

   - Clean browser profile
   - Load extension
   - Prepare sample data
   - Set browser zoom to 100%

2. **Capture screenshots:**

   - Take raw screenshots of each view
   - Capture at 1280x800 or larger
   - Save as PNG for quality

3. **Edit screenshots:**

   - Crop to exact dimensions
   - Add subtle borders or shadows
   - Add annotations if needed
   - Ensure consistent styling

4. **Review:**

   - Check all text is readable
   - Verify no personal info visible
   - Ensure professional appearance
   - Get feedback from team

5. **Export:**
   - Save as PNG
   - Optimize file size (use TinyPNG)
   - Name files descriptively
   - Keep originals for future edits

## Promotional Images

### Chrome Web Store Promotional Tiles

**Small Tile (440x280):**

- Extension icon + tagline
- Simple, clean design
- Brand colors
- Readable at small size

**Large Tile (920x680):**

- Extension icon + key features
- 3-4 feature highlights
- Professional design
- Clear call-to-action

**Marquee (1400x560):**

- Hero image with extension in action
- Main value proposition
- Feature highlights
- Professional photography or mockup

### Creating Promotional Images

Use Figma or Canva templates:

1. **Small Tile Template:**

```
Size: 440x280px
Background: Brand gradient or solid color
Icon: 128x128px centered
Tagline: "Save & Organize Social Bookmarks"
Font: Bold, readable at small size
```

2. **Large Tile Template:**

```
Size: 920x680px
Background: Brand colors with subtle pattern
Icon: 180x180px
Title: "Social Bookmarks Manager"
Features: 3-4 bullet points with icons
CTA: "Install Now" or "Get Started"
```

3. **Marquee Template:**

```
Size: 1400x560px
Background: Professional gradient or image
Left side: Extension screenshot or mockup
Right side: Value proposition + features
Logo: Top left or center
CTA: Bottom right
```

## Quality Checklist

Before submitting screenshots:

- [ ] All screenshots are correct dimensions
- [ ] Images are high quality (not blurry or pixelated)
- [ ] No personal information visible
- [ ] Consistent browser chrome (or cropped out)
- [ ] Text is readable at actual size
- [ ] Colors are accurate and professional
- [ ] No spelling errors in any text
- [ ] Annotations are subtle and helpful
- [ ] File sizes are under limits
- [ ] Files are named descriptively
- [ ] All required screenshots created
- [ ] Promotional images created (if needed)
- [ ] Team has reviewed and approved
- [ ] Screenshots match actual extension functionality
